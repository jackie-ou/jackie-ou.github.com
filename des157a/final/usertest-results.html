User Test Results and Plan

Users felt that there was a feeling of control when rolling the dice as opposed to pure luck. Players eventually figured out how to close the help window but it was not intuitive. The score's color and size make the text noticeable and much easier to read.

I plan to make the directions overlay's closing much more intuitive. I also want to brush up some minor inconsistencies within the CSS and button's wording. Finally, I want to make the overlay itself more polished so that it properly hides the underlying game.
